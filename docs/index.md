而月球上的一批先驱者——其中许多人正是当年在木星违规建设的底层劳动者（[基于蒸汽生活的人类](/settings/societies/steam_based_life_people.md)）决心将这场迫近的毁灭，转化为一场豪赌：他们计划利用撞击的惊天动能，将自己弹射向深空，以避免自己被资本剥削一辈子的命运。

这场被悄然促成的灾难，随即在人类社会中引爆了关乎生存、逃亡与责任的激烈争执。
